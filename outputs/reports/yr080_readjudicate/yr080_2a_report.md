# YR-080 2단계a — 교사 목적함수 재판정 (기준재 vs 사전식)

> 문헌 보정 시뮬레이션 조건. 교사=1800~5400s rolling-horizon rollout argmin. NN 학습 0.
> 교정 통계(표본sd÷n-1 + t분포). 원자료: `_ablation_refine`·`_ablation_horizon`·`_ablation_confound`.

## 한 줄 결론

**구 사전식(LEX) 회귀는 확정 기각.** 기준재 경제목적이 사전식으로는 **구조적으로 불가능한**
본선 보호를 (창이 충분할 때) 제공함을 15-seed 교정 CI로 확인. 단 배 보호는 트럭 꼬리(P95)
악화를 동반하므로 **P95 는 채택 게이트로** 관리한다(목적함수 안 아님, 사용자 결정 2026-07-22).

## 기전 (probe, 타이트 셀)

- 창 안 `vessel_delay`>0 = 0.26 (배가 이미 늦은 뒤 = **후행지표**)
- 창 안 `sts_wait`>0 = 0.60 (안벽크레인 유휴 = **선행지표**) · `transfer_wait`=0.00

## v1 결함과 교정 (스모크+8seed)

기준재 v1 을 교사에 그대로 얹으면 LEX 보다 트럭·배 **둘 다 악화**. 원인 2:
1. `ρ_crane=1.0` — 이동을 트럭대기와 동급으로 만들어 "일단 서비스" 규율 희석 + 이동시간
   이중계상(이미 대기항에 반영). → **crane_travel=empty_travel=0** (NUM_t0)
2. `sts_wait=0` — 창 안 유일 본선 선행지표 제거. → 참조 실험서 재활성(NUM_t0_stsN)

## 정직한 정정 (통계 버그)

초기 보고는 짝지은 CI 를 모표준편차(÷n)+z=1.96 으로 계산해 **과소**(n=8 ~29%)했고,
트럭 꼬리(P95)를 검정에서 누락했다. 표본sd(÷n-1)+t 로 교정하니 배 유의 셀 7→4(칼끝),
**트럭 P95 는 mid 부하에서 유의 악화(+5.6~8.8분)** = "mid 중립"이 아니라 순손실. 도구 수정
커밋(paired 교정 + d_p95_vs_LEX 신설). 적대적 검증 3관점(통계·항만운영·증류)이 지적, 직접 재계산 확증.

## 교락 해소 — 목적 덕 vs 창 덕 (15-seed, 같은 창 짝지음)

| 검정 | high-loose | high-tight | 판정 |
|---|---|---|---|
| 창 효과 LEX@3600−@1800 | −0.8 [−7.1,+5.5] | +0.8 [−25,+26.6] | 무의 |
| 창 효과 LEX@5400−@1800 | −5.0 [−11.9,+1.9] | −0.5 [−35,+34.5] | 무의 |
| 목적 @3600 (NUM−LEX) | **−16.2 [−25.1,−7.2]★** | **−39.6 [−64.7,−14.4]★** | 유의 |
| 목적 @5400 (NUM−LEX) | **−20.8 [−30.1,−11.5]★** | **−52.0 [−76.1,−27.9]★** | 유의 |

**LEX 는 창을 늘려도 배를 못 지킨다**(사전식이라 교환 불가). **같은 창에서 목적이 배를 지킨다.**
→ 배 보호는 **목적 재설계의 실제 효과**(창 아티팩트 아님). 목적은 창이 충분해야 발휘(상호작용):
high-loose 는 @1800 무의→@3600부터 발현, high-tight 는 마감 타이트로 @1800부터 발현.

## 트럭 꼬리(P95) 대가

배 보호가 강할수록(창 길수록) 트럭 P95 악화: 목적@5400 P95 high-loose **+6.5★**·high-tight **+5.2★**.
**예외**: STS대리@1800 은 high-tight 서 배 −27.2★·**P95 +0.7(무해)** — 꼬리 안 건드리고 배 지키는
지점(단 high-loose 는 배 약함). → 배↔꼬리 Pareto 상충. **P95 게이트가 작동점 선택의 결정 렌즈.**

## 판정 프레임 (확정)

- **목적함수** = 깨끗한 per-결정 스칼라 (배 보호 + 트럭 평균, 꼬리는 기존 `long_wait` 항).
- **합격 게이트** = 완주 100%·backlog 0·건전 믹스 + **트럭 P95 기준 대비 X분 초과 악화 금지(신설)**.
- **보고표** = 배 선석초과 + 트럭 평균 + **트럭 P95** 나란히.

## 미해결 / 후속 (2b)

1. **모든 수치가 교사층(NN 학습 0).** 배포되는 학생 NN 이 교사를 재현하는지 미측정 — 저장소
   자체 증류(yr073)에서 학생 붕괴 전례(top1_disagree 0.16~0.29, healthy=false). → **2b = 학생층
   bake-off**: 검증된 기준재 교사를 실제 증류해 학생층 선석초과·재현율·트럭 P95 순welfare 비교.
2. **긴 창 교사 ↔ 증류 학습성 상충**: pure@5400 는 배 최강이나 판별정보가 학생 관측 밖(미래
   rollout)이라 증류 격차 우려. STS대리@1800 은 신호가 관측 내(학습 용이)나 배 보호 약·이중계상.
   → 학생층에서 실측해야 A/B 결정 가능.
3. STS 대리항은 경제 provenance 부재·순환 튜닝 우려 → 채택 시 독립 자산비용으로 재정당화 필요.
