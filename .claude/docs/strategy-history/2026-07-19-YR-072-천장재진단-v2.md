## 3. 판정 — **가설 기각 (2차 확증): 헤드룸 부재는 문제 성질**

1. **예측 상금 0 은 프로파일·부하의 인공물이 아니다** — 문헌 보정·현실 부하·피크
   에서도 정확히 0. RL 의 예측 강점을 쓸 구조가 이 문제에 없음이 두 번째 독립
   조건에서 재현.
2. **탐색 상금은 커지기는커녕 소멸·역전** — mid −1.4%, high 는 8-seed 기준 없음.
   YR-070(6~9%)보다 오히려 작다. "혼잡할수록 조율 이득" 방향(YR-054)이 v2 고부하
   에선 관측 안 됨.
3. **창 기반 탐색(JR)의 고부하 붕괴 (+31.5, +15%)** — YR-068 의 차분 신호 급감
   (D_mean −5~−7→−1.6~−2.5)과 같은 서사: **결정 밀도가 높아지면 고정 시간창
   기반 방법이 약화**된다. 창 앵커(600s rollout·2400s 차분)가 규모 의존 병목.
4. 귀결: 학습 트랙(YR-066 차분 개량) 재개의 "부하 근거" 는 기각 — 남은 상위
   지렛대는 **YR-071 목적함수 재정렬** (cost↔health 정렬 후 진짜 상금 재측정)
   + (신규 서사) "문헌 보정 조건에서 규칙(SF-SPT)이 천장 근접으로 강하다"는
   결과 자체. 부수 검증: v2 지형에서도 SF_SPT > FIFO 서열 유지.
