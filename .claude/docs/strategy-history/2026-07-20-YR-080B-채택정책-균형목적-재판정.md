# YR-080 후보 B — 채택 정책(FT)의 균형 목적 재판정 (2026-07-20)

> 계기: 병렬 세션 YR-080 프로토타입이 "구 목적은 본선 방치"를 발견 → 프로토타입은
> 단순규칙 vs JR 만 봤음. 본 문서는 **채택본 FT** 를 균형 목적(LIT)으로 재채점.
> 진단(사전등록 아님)·재학습(④b)은 YR-080 정식 가중치 후. LIT 가중치=프로토타입 동일.

## 1. 질문·방법

FT 는 구 목적(계층 트럭우선, 본선 3순위)으로 학습됨. **본선을 공동 1차로 넣은 균형
목적으로 재채점하면 FT 의 "규칙 초과"가 살아남나.** 재학습 없이 FT/SF/JR 결정을
LIT RewardCalculator 로 채점. v2 본선스트레스 셀·12 seed·창 600s.

## 3. 판정 — FT 의 "규칙 초과"는 균형 목적서 사라진다

- **정상 부하(L80)**: 균형 목적으로 재면 **SF(5.74) < FT(5.89)** — 채택본이 규칙보다
  오히려 약간 나쁨. FT 는 트럭대기는 이기나(11.7<15.0) **본선을 2배 방치**(233.7 vs
  104.5)해 순비용이 상쇄·역전.
- **고부하(L112, FT 훈련범위 밖)**: FT 완주 0.84 로 붕괴·LIT총비용 27.4(SF 14.3의
  거의 2배) — OOD 취약.
- **본선 보호는 목적이 시킨다**: JR_LIT(균형 최적화)만 본선지연을 낮춤(114.9/230.7)
  vs 트럭우선(FT 233.7/273.3·JR_OLD 250.5/555.4). 목적 재정렬이 작동함을 확인.
- **한계(중요)**: 창 600s 는 근시(채택 교사 1800s·YR-078). 600s JR_LIT 가 SF 를 못
  이긴 것은 창 탓일 수 있음 → **1800s 균형 교사 상한은 YR-080c(④a)에서 별도 측정.**

## 4. 함의

- **채택본 FT 는 균형 목적서 유효하지 않다** (본선 방치 정책). 균형 목적으로 교사→
  증류→FT **재학습이 필수**(④b, YR-080 정식 후).
- 재학습의 상한(=증류할 가치)은 "1800s 균형 교사가 규칙을 이기나"에 달림 — YR-080c.
  만약 1800s 균형 교사도 SF 를 못 이기면, **균형 목적선 규칙이 near-optimal** 이라는
  프로젝트 반복 서사가 재현되는 것 (재학습해도 상한 낮음).
- 모든 수치 "미실측 임시 보정(assumed·프로토타입 LIT) 하" 단서. YR-080 정식 가중치서 재산정.

  규칙을 유의하게 초과.** 600s 재판정의 "SF 우위"는 창 근시 탓이었음(YR-078 정합) →
  **증류할 상한 존재 = ④b 재학습 정당화.**
- **정직한 경고 (중요)**: JR_LIT 는 LIT 총비용은 이기나 **본선지연이 SF 보다 오히려
  높다**(vessel_rush 175.9 vs 48.6·coincident 70.7 vs 39.1). 즉 프로토타입 LIT
  가중치가 "본선 보호"를 충분히 강제하지 못함 — 정규화가 본선지연의 한계비용을 낮게
  만들어, 교사가 트럭·효율로 이기며 본선을 흘려도 총비용이 낮다. **이대로 증류하면
  본선 안 지키는 정책을 배울 위험** → **YR-080 정식 가중치 정당화·민감도가 ④b 선결**.

## 6. 후보 B 종합 결론

1. **채택본 FT 는 균형 목적서 유효하지 않다** (본선 방치·규칙 미초과) → 재학습 필수.
2. **균형 교사(1800s)는 규칙을 이긴다** → 재학습에 상금 있음(증류 상한 존재).
3. **단 프로토타입 가중치는 본선 보호를 덜 강제** → **④b 재학습은 YR-080 정식(가중치
   정당화·본선 스트레스 검증) 확정 후.** 지금 재학습하면 잘못된 자로 배운다.
