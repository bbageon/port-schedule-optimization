## 1. 질문

**"G0 로 확인된 여지를 신경망이 빠르게 재현할 수 있는가"** — 느린 교사(JR_NEW,
결정마다 공동조합 전부를 1800s 가상실행)를 중앙 공동가치망 하나로 증류해, 현
행동공간에서 **학습 정책 최초의 SF-SPT 초과**와 교사 대비 계산시간 단축을 얻는다.

## 2. 교사 (동결 — YR-078 반영)

- `JointRolloutGreedy(RC, horizon_s=1800.0, objective=hierarchy_key, max_combos=64,
  base=SF-SPT)` — G0 승자 구성에서 창만 600→**1800s** (YR-078: 전 조건 안전
  단일값, 40분은 러시아워 ∩자 악화·적응형 v1 미채택). 지형 = v2 × mid(56)/high(80).

## 3. 학습 설계 (동결 — 조건 5·6 구현)

- **표본 (결정 단위)**: 크레인별 인코딩(itc-v4 `encode_observation`, state_norm P90
  — high train 선두 5 seed 로 재적합·박제) + 공동 feasible 조합 목록(각 조합의
  계층 tier 값) + **교사 선택 위치** + **SF-SPT 선택 위치** + 분기 플래그.
- **망**: 조합 입력 = [g, yc_A, queue_A, cand_A, yc_B, queue_B, cand_B] concat →
  MLP(256→128) → score (스칼라) + aux head(128→1). 실행 비교는 score 순위.
- **손실 = 순위 주목표 + advantage 보조 (조건 5)**: ① listwise CE — softmax(−score)
  가 교사 선택 조합을 분류 (= 방법 A 선택 모방, 조합 간 순위 학습) ② aux MSE 0.5 —
  tier-A(트럭대기) 창 비용을 회귀 (계층 1차 신호의 크기 감각). scalar 단일 회귀 없음.
- **분기 오버샘플 (조건 6)**: 교사≠SF-SPT 결정 가중 ×4. 분기 비율·분기 한정
  순위 일치율 보고 의무.
- **DAgger (조건 6)**: round-0 = 교사 궤적 60 에피소드(mid 740000~29·high
  740100~29) → v0 학습 → round-1 = **학생 실행** 궤적 20 에피소드(mid 741000~09·
  high 741100~09)를 교사로 재라벨 → 합산 재학습 v1. 학생이 만든 상태에서의 정답
  보강 — 한 번 실수 후 낯선 상태 붕괴 방지.
- 학습: Adam 1e-3·≤30 epoch·5 epoch 마다 val(743000~03 mid+743100~03 high, 8
  에피소드 학생 실행 평균대기) — **val-best 선택 (v0·v1 각각), 최종 학생 = val 우세
  라운드**. 탐험·TD 없음 (순수 지도).

## 4. 평가·판정 (동결)

- test: mid 742000~742019 · high 742100~742119 (수준별 20, 신규 대역). arms:
  SF_SPT · JR_NEW_1800(교사) · STUDENT(v0) · STUDENT(v1). paired bootstrap 10k
  (seed 75168), 절대차(분)+CI 보고.
- **G1 (primary)**: 최종 학생 vs SF_SPT Δ평균대기 CI 상한 < 0 — mid·high 둘 다.
- **G2 (보고)**: vs 교사 잔여 격차 CI + "근접" = 교사 이득(교사−SF 점추정)의 50%
  이상 확보 여부.
- **G3 (보고+명목)**: 에피소드 wall-time 학생 < 교사의 20% (기준선: 교사 600s 판
  5.3~8.9s/ep — 1800s 판은 본 실행에서 실측).
- **guard**: 완료율 전 seed 1.0·backlog 0·healthy mix·학생 REPO share ≤ 0.15·
  P95 유의 악화 없음·결정론(수준별 2 seed 재실행)·분기상태 리포트(비율·전체/분기
  top-1 일치율).
- **실패 조건**: G1 미달 → knob 재조정 없이 기각 박제 + 원인 분해(순위 정확도 vs
  분기 커버리지 vs 분포 이동) 보고. YR-074 진행 여부는 사용자 결정.
- 의미 제한: 문헌 보정 시뮬레이션 조건. 본선 보호 미검증(YR-041)·돌발 0(YR-077).
