# YR-077 사전등록 — 돌발(크레인 고장) 강건성 1단계 (2026-07-20)

> 실행 전 동결. 결과는 `## 실행 결과` 로 append. 사용자 승인 "응 진행해줘".
> 배경: FT 채택 전환 직후 — 전 실험이 돌발 0 세계였고, 채택본은 rollout 교사
> 증류라 예측 의존. 고장에서 이득 유지가 채택 유효성의 남은 관문 (YR-076 §5).

## 1. 조건 (동결)

- 주입: `TerminalGenParams.fault_outages` ∈ {0(대조), 1, 2} × 지속 900s(15분) —
  전용 스트림 `fault:{seed}` (기존 시나리오 draw 불변·기본 0 은 바이트 동일).
  크레인 정지 = 엔진 기존 EQUIPMENT_DOWN/UP 계약.
- 지형: v2 × calibrated mid/high. seed **신규**: mid 747000~747011·high
  747100~747111 (수준별 12, 셀 간 동일 seed — 차이는 주입뿐).
- arms: SF_SPT · STUDENT_FT(채택본 `student_ft.pt`). paired bootstrap 10k(75168).

## 2. 판정 (동결 — 진단형)

- **R1 (강건 유지)**: 전 주입 셀(수준×outage 1·2)에서 Δ평균대기(FT−SF) CI
  상한 < 0. 상한 ≥ 0 셀 존재 시 "부분 유지/붕괴" 로 셀별 정직 박제 — 구제 없음,
  후속(고장 포함 DAgger 재수집) 등록 제안만.
- guard: FT 완주율 ≥ SF −0.05 (동일 셀 상대)·정책단위 집계 mix 건강·결정론 1seed.
- 보고: outage 별 격차 곡선·완주율·P95. 문헌 보정 조건 한정.
