> 신용 희석 확정, BC 정책 56.25(SF_SPT +3.13, 종전 RL 대비 −20%)가 트랙 최고.

## 1. 질문 (성능 경로 + 2차 진단, 한 실험)

- **성능**: BC 시작점에서 TD 미세조정이 BC(56.25)를 넘어 SF_SPT(53.12) 초과에 닿는가.
- **진단**: 미세조정이 BC 를 되망가뜨리면(swa 붕괴 재발) "TD 신호가 건강한 정책조차
  유지 못 한다"는 신용 희석의 2차 확증이다 — 실패조차 정보다.

## 2. 알려진 위험 (실행 전 명시)

BC 의 Q 값은 cross-entropy 로 학습된 **서수(순위) 값**이고 비용 척도가 아니다. TD 는
Q 를 비용 척도(에피소드 잔여비용/cost_scale)로 재회귀시키므로, 미세조정 초기에 큰
손실·큰 gradient 가 순위 정보를 파괴할 수 있다("재척도화 파국"). **lr 사다리가 완충
장치**이며, lr 무관 전멸이면 신용 희석과 재척도화가 혼재된 것으로 보고 판정을
"파국 원인 미분리"로 낮춰 기록한다 (과잉 주장 금지).

## 3. 동결 설정

- 하네스: `src/yard_rl/experiments/yr062_bc_finetune.py` (`run_yr062`, full mode).
- 초기화: 커밋된 YR-061 phase-3 산출물 `outputs/reports/yr061_imitation/model_IMITATE.pt`
  를 warm-start — online·target 모두 BC 가중치, **optimizer 는 새로**(CE 단계 Adam
  모멘트 미승계), grad_steps=0.
- arm: **BC(동결 control)** + 미세조정 lr **{1e-4, 3e-4, 1e-3}**. 공통: ddqn·
  cost_scale=train 5-seed fit·γ 0.95 (YR-061 판정대로 기본 유지)·ε = 0.3/√ep
  (BC 근방 약한 탐험 — 표준 1/√ep 는 초기 완전 랜덤이라 BC 이점 소거).
- checkpoint 선택: **ep0(=BC 원본)을 후보에 포함**한 val 8-seed 실비용 최소 —
  미세조정이 순손해면 BC 로 회귀하는 것이 정직한 선택이고 그 자체가 판정 데이터.
- seed·시나리오·비교군·bootstrap: YR-061 과 동일 (train 600000~600149·val 610k·
  test 620k 20-seed·SF_SPT/FIFO·10k resamples) — BC/SF_SPT 와 paired 직접 비교.
- 계약 테스트: warm-start 가중치 승계·학습상태 초기화·형식 거부 3건 + 기존 7 passed.

## 4. 사전 판정 기준

- **F1 (유지)**: 선택된 미세조정 checkpoint 의 test swa ≥ 0.25 그리고 완료율 ≥ 0.95.
  ep0 이 선택되면 "미세조정 전 구간 순손해"로 기록 (F1 은 자동 충족되나 성능 기여 0).
- **F2 (성능, primary)**: best arm vs BC paired Δtotal — CI 상한 < 0 이면 **개선 확정**,
  0 포함이면 동급, 하한 > 0 이면 악화. vs SF_SPT 도 병기 — CI 상한 < 0 이면 "SF_SPT
  초과" (트랙 첫 사례가 됨).
- **F3 (진단)**: ep>0 checkpoint 들의 val swa 궤적 — 단조 붕괴면 신용 희석 2차 확증
  (§2 재척도화 혼재 주의 하에), 유지·개선이면 "TD 미세조정 가능" 증거.
- 노이즈 주의: 단일 학습 궤적·소형 시나리오 — 수치 몇 단위는 참고, 판정은 CI·swa 축.
