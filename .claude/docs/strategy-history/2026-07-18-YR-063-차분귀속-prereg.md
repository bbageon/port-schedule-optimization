## 1. 가설 H-063

크레인별 credit 을 팀 return 균등귀속 대신 **"내 행동이 팀 비용에 실제로 만든 차이"**
로 바꾸면 신용 희석이 제거되어, scratch 학습으로도 퇴화 없이 건강한 정책이 나온다.

## 2. 처방 (동결)

- **차분 credit**: `D_i = C_W(내 행동, 상대 행동 고정) − C_W(WAIT_i, 상대 행동 고정)`.
  C_W = 고정 시간창 **600s** 누적 팀비용(`_rollout_cost` — JR baseline 기계 그대로,
  분기 후 SF_SPT-resolver base_policy 진행). WAIT 선택 시 D=0 (자기 앵커). 음수 D =
  내 행동이 팀 비용을 줄임.
- **1-step 표본** (gamma_dt=0·부트스트랩 없음): YR-061 이분 검정 근거 — 관건은 후보 간
  순위 신호이고, TD 부트스트랩이 희석·불안정 통로였다. Q(s,c)→D 회귀, 실행은 argmin Q.
- **CTDE 지위 명시**: 학습 시 특권 정보(시뮬레이터 counterfactual rollout) 사용, 실행
  시 Q 만 — QMIX 의 centralized training 과 동일한 방법론적 지위. 평가 경로는 기존
  `run_episode`(QPreference) 그대로 (게이트·지표 계약 불변).
- 하네스: `src/yard_rl/experiments/yr063_diff_credit.py` (`run_yr063`, full mode).
  결정당 rollout 3회(실제 1 + 크레인별 counterfactual 2) — JR(≤64회/결정)보다 싸다.

## 3. 동결 설정

- 학습: ddqn·lr 1e-3·cost_scale=1.0(차분 credit 은 O(1) — fit 불요)·ε=1/√ep·
  초기화 seed 63,000·탐험 rng 63,100·150ep·checkpoint 15ep·val 8-seed 실비용 최소 선택.
- seed·시나리오: YR-061 과 동일 (train 600000~600149·val 610k·test 620k 20-seed).
- 비교군 (동일 test 대역의 기존 판정 행 **재사용** — yr013 reuse_jr 관례, 결정론 실증
  g0.95==pen0): CONTROL_TD(pen0, 70.11/swa 0.094)·BC(IMITATE, 56.25/0.491)·
  SF_SPT(53.12)·FIFO(65.43). paired bootstrap 10k (seed 75,061).

## 4. 사전 판정 기준

- **G1 (탈퇴화, primary)**: test swa ≥ 0.25 그리고 완료율 ≥ 0.95 — scratch 학습 최초의
  건강 정책이면 H-063 지지 (신용 희석 인과 3차 확증).
- **G2 (성능 사다리)**: paired Δtotal — ① vs CONTROL_TD CI 상한<0 (퇴화 RL 초과)
  ② vs BC CI 가 0 포함 이상 (모방 동급) ③ vs SF_SPT CI 상한<0 (**트랙 첫 규칙 초과** —
  학습이 규칙을 넘는 순간). 도달 최고 단계를 기록.
- **실패 조건**: swa < 0.25 → 차분 귀속으로도 순위 신호 부족 — 신용 희석 단독 원인
  가설 약화, 후보 대안(1-step 아닌 window 사다리·amortized-JR 회귀·BC+차분 혼합)을
  후속 등록하되 본 arm 은 기각으로 박제.
- 노이즈 주의: 단일 학습 궤적·소형 시나리오 — 판정은 CI·swa 축, 수치 몇 단위는 참고.
