# YR-014 — 채택 정책 locked 종합평가와 주장 확정

- **Epic**: Exp / **Priority**: 🟡 / **등록일**: 2026-07-12 / **재정의**: 2026-07-20
- **배경**: 현재 채택 후보는 중앙 공동가치망 `student_ft.pt`와 H1 목적지 규칙, 결정론적 안전 resolver다. YR-080·075-c에서 목적·목적지가 바뀌면 재학습한 후계 bundle을 평가한다. 과거 Candidate DQN·QMIX 중심 계획은 최종 채택 구조가 아니다.
- **목표**: 기준환경과 일반화 결과를 각각 봉인된 신규 test에서 한 번 평가하고, 증거가 허용하는 주장 범위를 최종 확정한다.

## 평가 묶음

1. 기준환경: 동결 정책 bundle 대 같은 목적지 규칙을 쓴 SF-SPT, 계획기 교사와 계산시간 비교
2. 조건 강건성: 부하·적재율·ETA 오류·고장·본선 마감·레인/이송 혼잡
3. 구조 일반화: YR-082 자료등급과 구조군을 붙여 YR-042의 stress·무재학습·조건부 재적응 결과를 별도 표로 수록
4. ablation: H1, 미세조정, DAgger, 정보 채널과 목적 계층의 기여 분리

## 수용 기준

- 모든 필수 seed 완주율 1.0, 종료 backlog 0, 안전·물리 위반 0
- 평균 대기와 P95 대기 개선 또는 사전등록 비열등
- 본선 완료·STS/이송 대기·레인 혼잡 악화 없음
- 이동·재조작·간섭과 행동분포 동반 보고
- 코드·schema·프로파일·정규화·체크포인트·seed manifest 재현

한 셀이라도 완주 guard가 깨지면 평균 대기 개선만으로 채택하지 않는다. 기준환경,
다른 구조 stress, 무재학습, 재적응 결과의 이름과 주장을 분리한다. Level 0~1은
개념시험, Level 2는 구조 이식, Level 3만 터미널별 시뮬레이션 성능 근거로 표시한다.

- **산출물**: locked 종합보고서, 재현 manifest, ablation·guardrail 표, 적용/보류 판정과 한계
- **의존**: YR-009 환경 동결, YR-080 목적 동결, YR-075-c 목적지 국소 헤드룸·정책 bundle 동결, YR-019 ETA 오류, YR-021 학습 밖 부하, YR-041 본선 마감, YR-082 자료등급, YR-083 런타임 호환, YR-042 구조 일반화 분류
- **범위 밖**: test 결과를 본 뒤 재튜닝, 실제 운영 로그 없이 현장 개선율 주장, 다중 블록 확장(YR-081)
