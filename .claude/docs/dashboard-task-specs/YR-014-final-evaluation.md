# YR-014 — Phase 9: 최종 평가·ablation·탄소 사후평가

- **Epic**: Exp / **Priority**: ⚪ / **등록일**: 2026-07-12
- **배경**: [03 §5](../../../docs/구현계획/03_정책_실험_평가.md) paired 통계(평균차·95% CI·효과크기·시나리오별 공개)와 §5.1 잠정 합격조건(안전위반 0·처리량 99%+·평균대기 5%+ 감소·P95 악화 없음 등). [03 §6](../../../docs/구현계획/03_정책_실험_평가.md) 탄소는 실측 계측근거가 있을 때만 정량화, 없으면 "감축 잠재력" 표현으로 제한.
- **목표(수용 기준)**: [05 §4 Phase 9](../../../docs/구현계획/05_테스트_로드맵_산출물.md) — **코드·설정·자료버전·seed 로 결과 재현 가능**. 봉인된 test 일자 1회 평가(재튜닝 금지), guardrail·실패사례·한계 보고 포함.
- **범위 밖(non-goal)**: 탄소를 근거로 한 정책 재선정, test 결과 기반 hyperparameter 수정.
- **계획**: test 일자 반복평가 → paired 통계·부하별 분석 → ablation(03 §3.3) → 사후 탄소 기대효과 → 최종 보고서.
- **산출물**: 최종 실험결과 보고서 (H1~H5 판정), 재현 실행법.
- **의존**: YR-011 (+ 진행된 경우 YR-012·013), YR-015 UI (검증 근거).
