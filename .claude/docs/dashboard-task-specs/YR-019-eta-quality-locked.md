# YR-019 — ETA 품질 조건 일반화 locked 시험

- **Epic**: Exp / **Priority**: 🟡 / **재정의**: 2026-07-20
- **배경**: 현재 정책은 주로 ±300초 ETA에서 개발됐고, 완벽 ETA 이득 0은 기존 행동공간에서만 확인됐다. 최종 목적지 결정이 바뀌면 ETA 가치도 달라질 수 있다.
- **목표**: 동결 정책 bundle이 현실적인 ETA 오차·편향·누락·노후 정보에서 규칙 대비 이득과 운영 guard를 유지하는지 평가한다.

## 조건

- `EMPIRICAL`, `BIASED`, `NO_SHOW`, `STALE`을 신규 paired seed로 평가한다.
- `PERFECT`는 정보가치 진단용 상한 arm일 뿐 배포 성능 주장에 넣지 않는다.
- 정책에는 각 arm에서 실제로 제공 가능한 ETA와 known-mask만 주고 실제 도착시각은 누출하지 않는다.
- 오류 크기·누락률·갱신주기와 사전등록 성능기준을 test 실행 전에 동결한다.

## guard

- 모든 seed 완주율 1.0·backlog 0·제약위반 0
- 평균·P95 대기와 본선·이송·레인 지표 비열등
- ETA가 나빠질수록 PRE_REHANDLE·REPOSITION·결정 wake가 퇴화하지 않음
- 정보품질별 계산시간·후보 수 보고

- **산출물**: ETA 품질별 paired 보고서, 누출검사, 실패 경계
- **의존**: YR-009 환경, YR-080 목적, YR-075-c 뒤 최종 정책 bundle
- **범위 밖**: test별 재학습, 완벽 ETA를 배포 입력으로 사용
