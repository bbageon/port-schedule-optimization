## 핵심 질문과 가설 (§1·§17)

> "차량 정보를 더 이른 시점에 확보할수록 야드크레인 작업순서 최적화 효과가 커지는가?"
> — 사전 결론이 아닌 **검증가설** (`Baseline < Exp-1 < … < Exp-4` 를 전제하지 않음).

- **H1**: 동일 제어범위에서 더 이른 정보 확보 → 외부트럭 대기비용 감소?
- **H2**: 부산항 제공 ETA 기반 사전 작업준비 > 게이트 진입 후 최적화?
- **H3**: 대기 감소가 이동·재조작·본선지연의 동반 개선으로 이어지는가?
- **H4**: 실제 공통 서비스영역의 다중 크레인 협조 > 크레인별 개별 최적화?
- **H5**: 운영 개선이 온실가스 감축 기대효과로 이어지는가?

## 실험 구성 (§3)

| 실험 | 정보 확보 시점 | 검증 목적 |
|---|---|---|
| Baseline | 현재 방식 (블록 도착순) | 기준선 |
| Exp-1 | 블록 진입 | 블록 내부 순서변경 효과 |
| Exp-2 | 게이트 진입 | 도착 전 선제 조정 효과 |
| Exp-3 | 사전 반출입정보 + 부산항 ETA | 사전정보 활용 효과 (선재조작·포지셔닝) |
| Exp-4 | (별도 축) 다중 YC 협조 | 협조 알고리즘 효과 |

실험요인 3축을 분리 식별: **정보 확보시점 × 제어범위(순서만/포지셔닝/선재조작) × 협조범위(단일/다중 YC)**.

## 비용·보상 구조 (§2.3)

- Yard Operation Cost = 외부트럭 누적대기 + 장기대기 패널티 + YC 이동 + 재조작 + 내부장비 대기 + 본선지연 패널티 + 간섭·불균형 + 계획변경 패널티 — 각 항을 Baseline 평균/SLA 로 **무차원화** 후 가중합. `Reward = -Cost`.
- 안전거리·비통과·장치가능 슬롯·도달영역은 비용항이 아니라 **Hard Constraint + Action Masking**.

## 고정 원칙

- **ETA 는 부산항 제공 외생 입력** — 예측모형·정확도 검증은 연구범위 밖 (§1·§18). ETA 오차 분석은 후속 민감도 항목.
- **탄소는 직접 최적화 안 함** — state/action/reward 미포함, 운영 개선의 사후 기대효과로만 해석 (§12.6·§13.4).
- **"배치"의 의미** = 물리 레일 재배치가 아니라 운영상 레인–YC 동적 할당·작업순서·재조작 선처리 (§1).
- **동일 통제조건** — 동일 물량·블록구조·장치상태·도착분포·처리속도·공통난수, 일자 단위 train/val/test (§15).

## 선행연구 위치 (§2.1·§19)

Kim/Lee/Hwang 2003 (학습 기반 YC 작업순서) · Guo 2011 (예측 도착정보 YC dispatching) · Fotuhi 2013 (YC Q-learning agent) · 황성범 외 2019 (부산 터미널 COPINO) · Ng 2005 (크레인 간섭) · He 2015, Papaioannou 2017 (효율↛에너지 자동 개선 — 탄소 사후해석 근거). 신규성은 사전정보 입력 자체가 아니라 **정보시점·정확도 효과의 분리 검증 + 서비스 수준 보호 + 다중 크레인 제약 반영**.
